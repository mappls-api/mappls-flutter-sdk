# Mappls Pin Strategy

## Add Marker
To add marker using Mappls Pin:
~~~dart
mapController.addSymbol(SymbolOptions(mapplsPin: mapplsPin))
~~~

## To set Camera to particular Mappls Pin

##### Sdk allows various method to Move, ease,animate Camera to a particular location :

~~~dart
mapController.moveCameraWithMapplsPin(CameraMapplsPinUpdate.newMapplsPinZoom('MMI000', 14));
mapController.easeCameraWithMapplsPin(CameraMapplsPinUpdate.newMapplsPinZoom('MMI000', 14));
mapController.animateCameraWithMapplsPin(CameraMapplsPinUpdate.newMapplsPinZoom('MMI000', 14));
~~~

## To set Camera to particular Mappls Pin  Bound

##### Sdk allows various method to Move, ease,animate Camera to a particular mappls pin Bound :

~~~dart
mapController.moveCameraWithMapplsPin(CameraMapplsPinUpdate.newMapplsPinBounds(['2T7S17', 'MMI000']));
mapController.easeCameraWithMapplsPin(CameraMapplsPinUpdate.newMapplsPinBounds(['2T7S17', 'MMI000']));
mapController.animateCameraWithMapplsPin(CameraMapplsPinUpdate.newMapplsPinBounds(['2T7S17', 'MMI000']));
~~~
